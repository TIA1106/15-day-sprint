# ✍️ TiaType Pro

TiaType Pro is a full-stack AI writing assistant created as part of this 15-day sprint challenge.

### 🔧 Features:
- Grammar correction  
- AI summarization  
- Keyword extraction  
- Tone detection  
- Autosave, voice input, and more
